# 5. General Comparative Overview

Full visualisations are available in the dedicated section:

[General Comparative Overview](https://doi.org/10.5281/zenodo.20682165)

In these comparative tables, values shown as “0%” do not necessarily reflect an actual absence of use. In some profiles, certain response options were not included in the original questionnaire. Since all profiles had an “Other (please specify)” field, any relevant practice could still have been reported. The lack of such indications means that “0%” should be interpreted as “no reported evidence” within the available data.

This section provides a general overview of how the nine professional profiles (Administration and Management profile is not included inside the comparative overview) compare across the main thematic blocks of the survey. The goal is not to repeat the detailed results already presented in the individual profile analyses, but to offer a consolidated picture of the sector and highlight a few transversal patterns that become visible only when the profiles are examined together. Zero values linked to non–applicable options are disregarded. The focus is intentionally selective: only differences or convergences that reveal meaningful sector–wide tendencies are discussed.
