# ARTEMIS-TWIN Survey Documentation

This section contains the ARTEMIS Task 3.4 User Survey Report and related visualisations.

The report presents the results of the ARTEMIS user survey carried out within Task 3.4 to investigate current digital practices, challenges, and training needs across the cultural heritage sector. It includes general findings, profile-specific analyses, cross-profile comparisons, training needs, and supporting visualisations.

The documentation is organised into the following sections:

- Introduction
- General Results
- Profile-Specific Results
- Training Needs and Preferences
- General Comparative Overview
- Appendix A
